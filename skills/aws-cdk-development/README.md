# AWS CDK Development Skill

AWS CDK best practices and patterns with the AWS CDK MCP server for infrastructure-as-code development.

## Installation

```bash
npx skills add https://github.com/aws-samples/sample-agent-skills-for-builders --skill aws-cdk-development
```

See [SKILL.md](./SKILL.md) for CDK patterns, pre-deployment validation, and the full reference material under `references/`.
