# AWS Agentic AI Skill

Bedrock AgentCore patterns for building and deploying production AI agents, including agent persistence, credential management, registry integration, and resource policies.

## Installation

```bash
npx skills add https://github.com/aws-samples/sample-agent-skills-for-builders --skill aws-agentic-ai
```

See [SKILL.md](./SKILL.md) for agent deployment patterns. The `cross-service/`, `references/`, `scripts/`, and `services/` directories contain detailed guides and automation.
