# AWS MCP Setup Skill

AWS Documentation MCP configuration for querying up-to-date AWS knowledge from AI agents.

## Installation

```bash
npx skills add https://github.com/aws-samples/sample-agent-skills-for-builders --skill aws-mcp-setup
```

See [SKILL.md](./SKILL.md) for the full configuration guide (Full AWS MCP Server and lightweight Documentation MCP).
