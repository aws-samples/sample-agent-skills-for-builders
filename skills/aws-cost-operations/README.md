# AWS Cost & Operations Skill

Cost estimation, monitoring, and operational excellence with AWS Pricing, Cost Explorer, and CloudWatch MCP servers.

## Installation

```bash
npx skills add https://github.com/aws-samples/sample-agent-skills-for-builders --skill aws-cost-operations
```

See [SKILL.md](./SKILL.md) for cost optimization patterns, observability guidance, and MCP server setup.
